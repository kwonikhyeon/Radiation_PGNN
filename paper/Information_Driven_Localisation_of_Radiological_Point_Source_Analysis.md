# Information Driven Localisation of a Radiological Point Source

## 논문 개요
- **제목**: Information driven localisation of a radiological point source
- **원본 파일**: Information_Driven_Localisation_of_Radiological_Point_Source.pdf

## 방사능 측정 방식
### 센서/검출기
- **주요 센서**: 모바일 방사선 측량 기기
- **검출기**: 감마 방사선 검출기 (선량당 계수/초)
- **지원 시스템**:
  - 차분 GPS (위치 측정)
  - 컴퓨터 (데이터 처리)

### 측정 모델
- **통계 모델**: 방사선 계수에 대한 포아송 통계
- **물리 법칙**: 역제곱 거리 법칙
- **측정 방정식**: `z ~ Poisson(λ)`
- **강도 관계**: `λ = Is·sk/d² + μb·sk`
  - Is: 소스 강도
  - sk: 노출 시간
  - d: 거리
  - μb: 배경 방사선

## 주요 기여점 (Contribution Points)
1. **결합 검출 및 파라미터 추정**: 베이지안 프레임워크에서 파티클 필터 사용
2. **Fisher Information Matrix(FIM) 기반 센서 제어**: 최적 관찰자 움직임을 위한 제어
3. **다중 스텝 전방 최적화**: 제어 벡터의 다중 스텝 전방 최적화 (관찰자 위치 및 노출 시간)
4. **Rao-Blackwellised 파티클 필터**: 효율적인 구현

## 실험 항목
### 실험 설정
- **소스 파라미터**:
  - 위치: (240m, 532m)
  - 강도: 18×10³ cts/s
- **탐색 영역**: 500m × 500m 직사각형 영역

### 알고리즘 파라미터
- **파티클 수**: 30,000개
- **검출 임계값**: P* = 0.75
- **SNR 임계값**: 12 dB
- **제어 최적화**: 1-3 스텝 전방 계획
- **안전 제약**: 최대 방사선 노출 제한

## 실험 결과
### 성능 지표
- **RMS 위치 오차**: ~8-10m
- **RMS 강도 오차**: ~5000-6000 cts/s
- **다중 스텝 전방 제어**: L=2,3에서 단일 스텝 대비 **~20% 관찰 스텝 수 감소**
- **필터 성능**: Rao-Blackwellised PF가 확장 파라미터 PF보다 우수한 성능

### 제어 전략
- **자율 제어**: 방사선 안전 제약 하에서 성공적인 자율 제어
- **관찰자 움직임**: 검출 후 추정 소스 위치 주변의 원형 움직임

## 현재 프로젝트와의 비교 적합도

### 성능 평가 지표 분석
1. **주행 거리**:
   - Fisher Information 기반 최적 경로
   - 다중 스텝 최적화로 효율적 거리 달성
   - 현재 PGNN 프로젝트의 RRT 기반 경로 계획과 비교 가능

2. **방사능 소스 탐지 정확도**:
   - 8-10m RMS 위치 오차
   - PGNN의 필드 복원 정확도가 더 우수할 가능성
   - 단일 포인트 소스에 특화된 정확도

3. **오염 회피 주행**:
   - 안전 제약 조건 포함
   - 최대 방사선 노출 제한
   - 현재 프로젝트의 위험 회피와 유사한 개념

### 비교 기준선으로서의 적합도
- **매우 적합**: 정보 이론 기반 센서 제어와 PGNN의 정보 획득 개념이 유사
- **장점**:
  - 베이지안 프레임워크 사용
  - 안전 제약 조건 고려
  - 실시간 제어 가능
- **제한사항**: 단일 포인트 소스만 처리

### 현재 프로젝트 대비 우위 확보 방안
- **다중 소스 처리 능력**: PGNN이 다중 소스 필드 복원에 우위
- **전체 필드 정보**: 단일 포인트 대신 전체 방사선 필드 제공
- **물리 유도 정확도**: 물리 법칙을 신경망에 직접 통합

## 구현 난이도
### 난이도: **높음 (4/5)**

#### 구현 가능한 요소
- 베이지안 추론 기본 구조
- 파티클 필터 알고리즘
- Fisher Information Matrix 계산

#### 구현 어려운 요소
- **복잡한 다중 스텝 최적화** 알고리즘
- **실시간 Rao-Blackwellised 파티클 필터** 구현
- **연속 제어 공간에서의 최적화**
- **안전 제약 조건이 포함된 제어 시스템**

#### 필요 기술
- 고급 베이지안 추론
- 최적 제어 이론
- Fisher Information 이론
- 실시간 최적화 알고리즘
- 파티클 필터링 전문 지식

## 결론
본 논문은 현재 PGNN 프로젝트와 이론적으로 매우 유사한 접근을 보인다. 특히 정보 이론 기반의 센서 제어와 안전 제약 조건 고려 측면에서 직접적인 비교가 가능하다. PGNN 기반 시스템이 단일 소스에서 다중 소스로, 포인트 추정에서 필드 복원으로 확장된 버전으로 우위를 보일 수 있다.
